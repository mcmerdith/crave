# Crave API

API server for the Crave mobile app

# Getting Started

### Install the Vercel CLI

```
npm i -g vercel
```

### Link the project

```
vercel link --repo
```

### Follow prompts from the CLI

- Authorize with Vercel in browser
- Link project `mcmerdith-development/crave-api`
  - MAKE SURE YOU DO NOT CREATE A NEW PROJECT
  - If you haven't updated your remote to [https://github.com/mcmerdith/crave.git](https://github.com/mcmerdith/crave.git) make sure you do so

### Pull the environment

```
vercel env pull
```
